#Arquivos e configurações necessárias para os notebooks.
#Checklist dos equipamentos.
##Windows 7
1. Atualizar para SP1
2. Instalar chrome
3. Instalar ARDUINO IDE
4. Baixar CADESimu 4
5. Criar atalho do CADESimu na área de trabalho
6. Instalar Veyon
7. Instalar e configurar BGInfo
8. Colocar BGInfo na inicialização
9. Instalar .netFramework 4.0
10. Instalar Python 3.8
11. Instalar driver CP210x
12. Instalar driver CH340
13. Configurar Veyon
14. Pegar MAC Address e patrimônio do equipamento
15. Instalar atualizações
16. Conferir se está tudo funcionando
17. Instalar esp32 no Arduino IDE - versão 1.0.6 (nos instalados remover o novo e instalar o antigo)
